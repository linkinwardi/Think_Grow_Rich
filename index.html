The premise

The book opens with a story: a retail team spent two days hand-comparing a competitor's price spreadsheet against their own. A short Python script did the same work in minutes. That gap — between repetitive human effort and a few dozen lines of code — is the entire thesis.

This companion treats the book as two movements. Part I builds the language itself: data types, flow control, functions, and the collection types that hold real-world data. Part II turns that language outward, onto files, spreadsheets, PDFs, the web, email, images, and the operating system itself.

How to use this site

  • Chapters — eighteen concise summaries, each with learning objectives, key constructs, a worked code sketch, and a schematic where the concept is structural.
  • Study Materials — annotated schematics, two step-by-step project tutorials, and a glossary of the vocabulary the book assumes.
  • Quiz Engine — an adaptive question bank with four modes, immediate rationale on every answer, and a graded debrief.
